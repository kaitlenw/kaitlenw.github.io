Ultimate Tic Tac Toe

Back To All Projects

An implementation of the extended version of the well-known Tic Tac Toe.

In this version, you play to win three boards in a row instead, made more difficult by the fact that your opponent's last move dictates your next move! In addition to building the functionality to validate moves and determine when the game has been finished (in any capacity), this implementation also features an AI player, produced by using Monte Carlo Tree Search.

This project is written in Python 3, and makes use of the PyQt library for the GUI. The code can be found here.
