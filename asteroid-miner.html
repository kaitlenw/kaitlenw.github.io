A clone of Asteroid Miner; gameplay of the orginal game can be viewed here.

In Asteroid Miner, you control a spaceship, to fly around the galaxy, destroying asteroids and collecting the materials found within them. Selling these materials at a nearby planet allows you to replenish fuel, make repairs to the ship and purchase other upgrades.

This project is written in C#, using the Unity game engine, and graphics mostly from Kenney.nl. The game is currently in the early stages of development, and this page will be updated with more development details as it progresses. In the future, screenshots of the game will also be added. The current state of the game code can be viewed on Github here.
