An implementation of the well-known board game, Monopoly, using Java.

Current Functionality

Since the implementation is still in development, it is not yet possible to fully play the game. Currently, the following functionality has been implemented:

  • GUI:
    • Displays the board and current player positions on the board.
    • Displays information about the current player - their name, current location, whether they are in jail and their current balance.
    • Game log keeps a record of past events in the game.
  • Player movement:
    • Rolling the dice and adjusting the player's position based on the numbers.
    • Taking another turn after rolling a double.
    • Going to jail if three doubles have been rolled in a row, or if the player lands on the Go To Jail square.

The next goals for the project are buying properties, paying rents from landing on owned properties and using Chance and Community Chest cards.

Technical Details

The game is written in Java, using JavaFX for the GUI. It is also backed by SQLite to store information about the board (including its positions and Chance and Community Chest cards), and the players (including their current position, balance and properties they've bought and built houses or hotels on).
